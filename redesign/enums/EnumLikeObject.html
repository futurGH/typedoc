Enumeration EnumLikeObject

Since TypeScript's enum can be inconvenient to work with, some packages define their own enum-like objects:

export const EnumLikeObject = {
    Pending: 'pending',
    InProgress: 'inProgress',
    Completed: 'completed'
} as const
